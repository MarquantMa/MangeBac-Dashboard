ManageTask Student Dashboard - Web Application
===================================================

System Requirements:
1. Python 3.x installed on your computer.
2. Install required libraries by running in your terminal or command prompt:
   pip install requests beautifulsoup4

How to Launch:
- On Windows: Double-click 'RunDashboard.bat'
- On macOS/Linux: Open your terminal, navigate to this folder, and run:
  chmod +x RunDashboard.sh
  ./RunDashboard.sh
