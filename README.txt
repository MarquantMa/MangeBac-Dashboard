ManageTask Student Dashboard - Web Application
===================================================

System Requirements:
1. Python 3.x installed on your computer.
2. Install required libraries by running in your terminal or command prompt:
   pip install -r requirements.txt

How to Launch:
- On Windows: Double-click 'RunDashboard.bat'
- On Linux: Open your terminal, navigate to this folder, and run the user startup installer:
  chmod +x RunDashboard.sh
  ./RunDashboard.sh
  The backend will be configured to start for the current user at login.
  Open http://localhost:8082 in your browser.
